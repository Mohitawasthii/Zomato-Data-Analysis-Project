## 📌 Project Overview

This project performs an end-to-end **Exploratory Data Analysis (EDA)** on the **Zomato Restaurant Dataset** to uncover valuable insights into restaurant ratings, cuisines, pricing, online delivery, table booking, customer preferences, and location-based trends. The analysis demonstrates how data can support strategic decisions in the food delivery and restaurant industry.

## 🎯 Project Objectives

- Analyze restaurant and customer preference data.
- Perform data cleaning and preprocessing.
- Explore rating, pricing, and cuisine trends.
- Study online delivery and table booking behavior.
- Visualize restaurant performance across locations.
- Generate actionable business insights.

## 📂 Dataset Features

The dataset includes restaurant information such as:

- Restaurant Name
- City
- Country
- Cuisines
- Average Cost for Two
- Currency
- Aggregate Rating
- Votes
- Online Delivery
- Table Booking
- Price Range
